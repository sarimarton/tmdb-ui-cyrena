# tmdb-ui-cyrena

A simple, purely functional/reactive TMDb UI using Cyrena.

#### Install & Run:
```
git clone https://github.com/sarimarton/tmdb-ui-cyrena.git
cd tmdb-ui-cyrena
npm i && npm start
```
