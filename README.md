# tmdb-ui-cyclejs

A simple, purely functional/reactive TMDb UI using Cycle.js.

#### Install & Run:
```
git clone https://github.com/sarimarton/tmdb-ui-cyclejs.git
cd tmdb-ui-cyclejs
npm i && npm start
```
