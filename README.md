# tmdb-ui-powercycle

A simple, purely functional/reactive TMDb UI using Powercycle.

#### Install & Run:
```
git clone https://github.com/sarimarton/tmdb-ui-powercycle.git
cd tmdb-ui-powercycle
npm i && npm start
```
